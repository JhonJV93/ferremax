# Ferremax Ecuador
Sistema de gestión de sucursales, productos y permisos — Proyecto Semana 8

## Tecnologías
- Next.js 16
- Prisma ORM
- PostgreSQL
- Autenticación JWT
- Caché Cache-Aside

## Funcionalidades implementadas
✅ Autenticación y autorización por roles
✅ Optimización de consultas sin N+1
✅ Caché para reducir carga en base de datos
✅ Gestión de sucursales y organizaciones
